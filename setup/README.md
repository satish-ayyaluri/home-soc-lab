
# Setup Steps

1. Installed Ubuntu and Kali Linux in VirtualBox.
2. Installed and configured Snort IDS on Ubuntu.
3. Configured HOME_NET.
4. Created an ICMP detection rule.
5. Verified Snort alert generation.
6. Installed Splunk Enterprise.
7. Added Snort alert log as a Splunk data input.
8. Verified successful log ingestion.
9. Created dashboard panels for Total Alerts, ICMP Alerts, and Alert Timeline.
